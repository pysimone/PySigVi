Introduction

Introduction

PySigVi is a GUI (Graphical User Interface) for GNU Privacy Guard (GPG).

It checks the detached signatures of a file.

Under the hood it repeat:

gpg --verify detached_signature file_to_verify

for each detached signature.

PySigVi does not create the signatures! It only checks the existing ones created with some other software.

To work properly, all the Prerequisites must be met.

Index

Prerequisites

GNU Privacy Guard installed and in the path of your system.

If you want to execute as a Python3 script, you also need:

  • Python3 installed.
  • Pyside6 for the GUI (Graphical User Interface). It should be installed in your system or virtual environment.

Installation and Execution

For python users

If you want to execute PySigVi as a python3 script, download in a directory these files and folders:

  • pysigvi.py
  • docs
    • about.html
    • license.html
    • manual.html
  • icons
    • blue-folder-open-document.png
    • blue-folder-open-document-text.png
    • door-open-out.png
    • pysigvi.ico
    • ui-check-box.png

Then execute the script:

$ python3 pisigvi.py

For Windows User

If not already installed, download and install GPG.

Download the latest executable pysigvi.exe

from this url: https://github.com/pysimone/PySigVi/releases/latest .

Double click on pysigvi.exe to run the script.

For Linux User

If not already installed, install GPG:

sudo apt-get install gnupg

Download the latest binary pysigvi

from this url: https://github.com/pysimone/PySigVi/releases/latest .

Make the file executable:

chmod a+x pysigvi

Execute the binary:

./pysigvi

Or install the binary in the system:

sudo install ./pysigvi /usr/bin/

Execute the binary:

pysigvi

Instructions

You need to open the file to be verified and the signatures to verify.

Open the file to verify:

Click on the button with the three dots "...".

Or from the menu: File->Open File.

Or on the tools bar click the first icon.

Or use the shortcut: Ctrl+F.

Now you can select the file.

In the dialog select a file. The name will appear in the "Name" field.

Click the "Open" button to confirm the selection. Or click the "Cancel" button to go back to the main window with no selection.

The selected file, with its complete url, now appears in the "File to Verify" field.

Select the signatures:

Click on the button "Add Signatures".

Or from the menu: Tools->Add Signatures.

Or from the tools bar click the second icon.

Or use the shortcut: Ctrl+S.

Now you can select one signature with a click or multiple signatures holding Ctrl and clicking on the files.

Click the "Open" button to confirm the selection. Or click the "Cancel" button to go back to the main window with no selection.

The selected file or files, with their complete url, now are listed in the "Signatures" field.

You can add more signatures repeating the previous steps.

Remove the signatures:

If you want to clear the "Signatures" field:

Click on "Remove Signatures".

Or from the menu select: Tools->Clear Signatures.

If the "File to verify" and the "Signatures field" are not empty, you can check the signatures of the file.

Check the signatures:

Click on the button "Check Signatures".

Or from the menu: Tools->Check Signatures.

PySigVi performs the check of the file with the signatures.

In the field "Check" is reported the result of the check.

If a signature is verified its row is green.

If a signature is NOT verified its row is red.

If one ore more signatures are not verified an alert dialog appears.

Acknowledge the alert by clicking on the "Ok" button.

In the rows are written some notes about the result of the check.

After reading the notes you can decide to undertake more actions about the signatures.

You can repeat the check repeating the steps above.

Clear the results:

Click on the "Clear Check" button.

Or from the menu select: Tools->Clear Check.
